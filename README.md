# MMP Icon Mobile App


## Overview
Simple application showing the latest use of using ionic framework webcomponents with vuejs. 

- vue-router
- vuex
- VeeValidate
- @ionic/vue

Ionic v4 Documentation
- https://beta.ionicframework.com/docs/
